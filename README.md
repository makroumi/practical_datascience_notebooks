# Practical Data Science Notebooks
### A minimal collection of hands-on, runnable Jupyter notebooks teaching NumPy, pandas, Matplotlib, and introductory machine learning.
---

## Quick summary

- **Audience:** aspiring data scientists and engineers who learn by doing.  
- **Goal:** compact, deterministic notebooks that run quickly and illustrate common workflows (explore → transform → visualize → model).  
- **Repo focus:** notebooks only, files small, outputs cleared.
